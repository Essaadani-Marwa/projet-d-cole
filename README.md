# Système de Gestion Académique

Application web pour gérer un établissement universitaire.

## Fonctionnalités
- **3 types d'utilisateurs** : Admin, Enseignant, Étudiant
- **Gestion complète** : étudiants, enseignants, matières, notes
- **Sécurisé** : connexion par mot de passe

## Technologies utilisées
- PHP
- MySQL 
- HTML/CSS
- JavaScript

## Installation
1. Installer XAMPP
2. Copier le dossier dans `htdocs`
3. Démarrer Apache et MySQL
4. Importer la base de données
5. Aller sur `http://localhost/projet-d'ecole/login.php`
